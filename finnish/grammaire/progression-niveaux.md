# Progression des Lecons par Niveau

*Structure detaillee pour l'application d'apprentissage*

---

## Niveau A1 - Debutant (environ 60-80 heures)

### Module A1.1 - Premiers pas (Ensimmaset askeleet)

**Lecon 1 : Salutations et presentations**
- Hei, moi, terve, moikka
- Mika sinun nimesi on? Mina olen...
- Mita kuuluu? Hyvin menee / Kiitos hyvin

**Lecon 2 : L'alphabet et la prononciation**
- Voyelles : a, e, i, o, u, y, a, o
- Consonnes et sons speciaux
- Regles de prononciation (chaque lettre se prononce)

**Lecon 3 : Les chiffres 0-20**
- nolla, yksi, kaksi, kolme, nelja, viisi...
- Exercices de comptage

**Lecon 4 : Jours et mois**
- maanantai, tiistai, keskiviikko...
- tammikuu, helmikuu, maaliskuu...

**Lecon 5 : Le verbe olla (etre)**
- mina olen, sina olet, han on
- me olemme, te olette, he ovat
- Questions avec -ko : Oletko suomalainen?

---

### Module A1.2 - Vie basique (Peruselama)

**Lecon 6 : Verbes du type 1**
- puhua, lukea, kirjoittaa, opiskella
- Conjugaison au present
- Introduction a la gradation consonantique

**Lecon 7 : Le partitif - Introduction**
- Apres les nombres : kaksi koiraa
- Phrases negatives : Minulla ei ole koiraa
- Verbes partitifs : rakastan sinua

**Lecon 8 : Possession avec "minulla on"**
- Minulla on koira / Minulla ei ole koiraa
- Sinulla on, hanella on...

**Lecon 9 : Les cas locatifs - Ou?**
- Missa? -ssa/-ssa : Olen talossa
- Mista? -sta/-sta : Tulen talosta
- Mihin? -Vn/-hVn : Menen taloon

**Lecon 10 : Le genitif basique**
- Possession : koiran nimi
- Avec postpositions : talon edessa

---

### Module A1.3 - Conversations simples (Yksinkertaiset keskustelut)

**Lecon 11 : La famille**
- perhe, aiti, isa, veli, sisko
- isovanhemmat, lapset
- Minun perheessani on...

**Lecon 12 : La nourriture**
- leipa, juusto, maito, kahvi, tee
- Verbes : syoda, juoda
- Haluatko kahvia?

**Lecon 13 : La maison**
- talo, asunto, huone, keittio, makuuhuone
- Meubles basiques
- Asun kerrostalossa

**Lecon 14 : Les couleurs et adjectifs**
- punainen, sininen, vihrea, keltainen
- Accord des adjectifs
- iso, pieni, kaunis, hyva

**Lecon 15 : Imperatif et verbes modaux**
- Tule tanne! Mene pois!
- haluta, voida, osata, saada
- Haluaisin... (politesse)

---

## Niveau A2 - Debutant avance (environ 80-100 heures)

### Module A2.1 - Autonomie quotidienne

**Lecon 16 : Tous les types verbaux**
- Type 2 : syoda, juoda (syön, juon)
- Type 3 : tulla, menna (tulen, menen)
- Type 4 : haluta, tavata (haluan, tapaan)
- Type 5 : tarvita, valita (tarvitsen, valitsen)

**Lecon 17 : Les cas locatifs exterieurs**
- Missa? -lla/-lla : Olen torilla
- Mista? -lta/-lta : Tulen torilta
- Minne? -lle : Menen torille

**Lecon 18 : Le partitif approfondi**
- Tous types de mots
- Verbes avec partitif : odottaa, etsia, rakastaa
- Quantites indefinies

**Lecon 19 : Postpositions**
- edessa, takana, vieressa, alla, paalla
- kanssa, ilman, jalkeen, aikana
- Avec genitif : talon edessa

**Lecon 20 : Le conditionnel - Politesse**
- Haluaisin kahvia
- Voisitko auttaa?
- Olisi kiva...

---

### Module A2.2 - Raconter et decrire

**Lecon 21 : L'imparfait**
- Formation : -i- + terminaison
- puhuin, puhuit, puhui
- Gradation consonantique a l'imparfait

**Lecon 22 : Le parfait**
- olen + participe passe (-nut/-nyt)
- Olen asunut Suomessa
- Difference avec l'imparfait

**Lecon 23 : Exprimer des opinions**
- Minusta, mielestani
- Luulen etta... Uskon etta...
- koska, vaikka, mutta

**Lecon 24 : Le 3e infinitif**
- Menen nukkumaan
- Olen lukemassa
- Tulen syomasta

**Lecon 25 : Le passif - Introduction**
- Suomessa puhutaan suomea
- Mennaan! Syodaan!
- Passif = "on" en francais

---

## Niveau B1 - Intermediaire (environ 100-150 heures)

### Module B1.1 - Communication effective

**Lecon 26 : Comparatif et superlatif**
- -mpi : isompi, kauniimpi
- -in : isoin, kaunein
- Irreguliers : hyva-parempi-paras

**Lecon 27 : L'essif et le translatif**
- Essif (-na/-na) : Opiskelen laakarinä
- Translatif (-ksi) : Tulen laakariksi
- Expressions temporelles

**Lecon 28 : Le plusquamperfait**
- olin + participe passe
- Olin jo syonyt kun...
- Sequences d'evenements

**Lecon 29 : Verbes de perception**
- nayttaa, tuntua, kuulostaa, maistua
- + ablatif : Naytat vasyneelta
- Sensations et impressions

**Lecon 30 : Le pronom relatif joka**
- Mies, joka asuu taalla
- Kirja, jonka ostin
- Auto, jossa istumme

---

### Module B1.2 - Expression nuancee

**Lecon 31 : Le partitif pluriel**
- Formation complexe
- koiria, taloja, ihmisia
- Alternances de radical

**Lecon 32 : Les cas au pluriel**
- Genitif pluriel : koirien, talojen
- Locatifs pluriels : taloissa, taloista, taloihin
- Patterns reguliers

**Lecon 33 : Le passif aux temps passes**
- Imperfait passif : puhuttiin
- Parfait passif : on puhuttu
- Conditionnel passif : puhuttaisiin

**Lecon 34 : Rections verbales**
- Verbe + cas specifique
- pitaa + elatif : Pidan sinusta
- uskoa + illatif : Uskon sinuun

**Lecon 35 : Le 4e infinitif**
- Tekeminen on hauskaa
- Nominalisations
- Puhuminen, kirjoittaminen

---

## Niveau B2 - Intermediaire avance (environ 150-200 heures)

### Module B2.1 - Maitrise structurelle

**Lecon 36 : Le potentiel**
- -ne- : tullen, puhunnen
- Probabilite : Han tullee huomenna
- Usage litteraire et formel

**Lecon 37 : Le participe agent**
- -ma/-ma + possessif
- Hanen kirjoittamansa kirja
- Structures complexes

**Lecon 38 : Constructions temporelles**
- -essa/-essa : syodessa
- -tuaan/-tyaan : syotyaan
- Equivalents de "quand/apres"

**Lecon 39 : Lauseenvastikkeet**
- Structures remplacant les subordonnees
- Economie de langage
- Style ecrit formel

**Lecon 40 : Cas rares**
- Comitatif : vaimoineen
- Instructif : kaksin kasin
- Abessif : rahatta

---

### Module B2.2 - Perfectionnement

**Lecon 41 : Referatiivirakenne**
- Rapporter les paroles
- Han sanoi tulevansa
- Structures indirectes

**Lecon 42 : Formation des mots**
- Prefixes et suffixes derivationnels
- -ja/-ja : opettaja, kirjoittaja
- -uus/-yys : kauneus, rakkaus

**Lecon 43 : Registres de langue**
- Formel vs informel
- Choix stylistiques
- Contextes d'usage

**Lecon 44 : Finnois parle avance**
- Toutes les transformations
- Expressions idiomatiques
- Comprehension native

**Lecon 45 : Revision integree**
- Tous les cas en contexte
- Tous les temps et modes
- Production ecrite et orale

---

## Themes transversaux (toutes lecons)

### Vocabulaire thematique
- A1 : famille, maison, nourriture, corps, ville
- A2 : travail, sante, loisirs, voyages, achats
- B1 : actualites, culture, opinions, sentiments
- B2 : abstrait, professionnel, litteraire

### Finnois parle (integre des A2)
- Formes courtes des pronoms
- Passif pour "nous"
- Contractions courantes
- Expressions familieres

### Exercices types par niveau
- A1 : QCM, association, repetition
- A2 : Traduction, completement, mini-dialogues
- B1 : Redaction guidee, comprehension, reformulation
- B2 : Production libre, analyse, debat
